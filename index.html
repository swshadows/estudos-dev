Estrutura do projeto

    📁 Estudos Web

  • 📁 HTML5 - HyperText Markup Language
    • 🌐 Básico: Textos, listas, imagens e tabelas;
    • 🌐 Intermediário: Formulários, inputs, selects e textarea
    • 🌐 Avançado: Audio, video, iframe, animações mixado com um pouco de JS básico
  • 📁 CSS3 - Cascading Style Sheets
    • 🌐 Básico: Paragrafos, tamanhos, fontes, margin, padding e etc;
    • 🌐 Intermediário: Display, overflow, floating, alinhamentos, media queries e etc;
    • 🌐 Avançado: Pseudoelementos, calc, seletores, transform, flexbox, grid e etc;
  • 📁 JS - JavaScript
    • 🌐 Básico: Variáveis, operações, condições, repetições, funções, datas, construtores e etc;
    • 🌐 Intermediário: DOM, EventListener, 'this', mousein/out, Math, JSON, localStorage, interval e timeout, e etc;
    • 🌐 Avançado: This e Bind, fetch, promise, AJAX Async Await, Filter, Map, Reduce, Try/Catch, e etc;
  • 📁 PHP 7.4 - HyperText Preprocessor (requer servidor)
    • 🌐 Testes iniciais: Testes iniciais com a linguagem ;
    • 🌐 Processamento de dados: Forms e arquivos;
    • 🌐 Classes e POO: Uso de programação orientada a objetos com PHP;
    • 🌐 Composer: Fazendo uso do gerenciador de pacotes Composer;
    • 🌐 Banco de dados: Testes iniciais com banco de dados MariaDB;
    • 🌐 SOLID: Aprendizado do conceito SOLID;
    • 🌐 Senhas: Fazendo uso da encriptação de senhas;
    • 🌐 Imagens: Processando e manipulando imagens;
    • 🌐 Datas e tempo: Utilizando datas e tempos mais a fundo;
    • 🌐 API: Programando minha API com banco de dados;
    • 🌐 MVC: Aprendizado do padrão MVC (Não concluído);
  • 📁 Frameworks Web
    • 🌐 Bootstrap: Utilizando Bootstrap 5;
    • 🌐 jQuery: Utilizando jQuery;
  • 📁 Projetos-Copy: Projetos criados com uma base;
  • 📁 Projetos-Zero: Projetos criados do absoluto zero;
